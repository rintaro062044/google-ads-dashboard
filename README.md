# Google広告 分析ダッシュボード

Google広告の管理画面からエクスポートしたExcelレポート（`〇〇_KW_...` / `〇〇_クエリ_...` / `〇〇_広告_...` の3シート構成）を読み込み、ブラウザ上だけで下記の分析を行うツールです。

- KPIサマリー（目標CPAとの比較つき）
- キャンペーンタイプ分析（タイプ軸／月別軸／競合比較軸）
- キャンペーン分析（キャンペーン軸／月別軸）
- キーワード分析（品質スコア・コンバージョン0円消化・掲載順位起因の機会損失）
- TD分析（広告見出し・説明文の訴求軸を自動分類し、競合と比較）
- アカウント改善ポイントの自動診断

## 使い方

1. https://rintaro062044.github.io/google-ads-dashboard/ を開く
2. 「Excelを読み込む」から対象のExcelファイル（.xlsx）を選択する
3. 分析対象（メインサービス）を選び、必要に応じて目標CPAを入力する

## データの扱いについて

すべての集計処理はブラウザ内のJavaScriptで完結します。読み込んだExcelファイルの内容がサーバーに送信されることはありません。

## 動作環境

`DecompressionStream` API を使用してxlsx（ZIP形式）を解凍しているため、Chrome / Edge などの比較的新しいChromium系ブラウザでの利用を推奨します。

## ローカルでの確認方法

サーバー不要の静的HTML1枚の構成です。`index.html` をブラウザで直接開くだけで動作します。
